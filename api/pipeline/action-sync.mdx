The `ActionSyncEngine` class maps IDE actions from the generated script to exact millisecond timestamps by matching trigger words with the Whisper transcription. This enables frame-perfect synchronization between narration and code actions.

## Class Overview

#### Example Usage

```python
from pathlib import Path
from pipeline import (
    ScriptGenerator,
    AudioGenerator,
    TimestampExtractor,
    ActionSyncEngine
)

# Generate script and audio
script_gen = ScriptGenerator()
script, _ = script_gen.generate("Create a React component")

output_dir = Path("./output")
audio_gen = AudioGenerator(output_dir)
audio_path, _, _ = audio_gen.generate_audio(script)

# Extract timestamps
ts_extractor = TimestampExtractor()
timestamps, _ = ts_extractor.extract_timestamps(audio_path)

# Synchronize actions
sync_engine = ActionSyncEngine(output_dir)
synced_actions, sync_latency = sync_engine.sync_actions(script, timestamps)

print(f"Synchronized {len(synced_actions)} actions in {sync_latency:.0f}ms")

# Display synchronized actions
for action in synced_actions:
    print(f"{action.timestamp_ms}ms: {action.type} ('{action.trigger_word}')")
```

## Synchronization Algorithm

The sync engine uses a sequential matching algorithm:

1. **Extract all actions** from script chunks in order
2. **For each action:**
   - Clean the trigger word (lowercase, strip)
   - Search timestamps for matching word
   - Match must occur after the last matched timestamp (prevents duplicates)
   - Strip punctuation from transcribed words (.,!?;:'")
3. **Handle unmatched triggers:**
   - Log warning if trigger word not found
   - Continue processing remaining actions

### Matching Logic

```python
# Exact match
trigger: "create" → transcribed: "create" ✓

# Case-insensitive
trigger: "Create" → transcribed: "create" ✓

# Punctuation stripped
trigger: "create" → transcribed: "create," ✓

# Partial match (fallback)
trigger: "function" → transcribed: "functions" ✓
trigger: "type" → transcribed: "typing" ✓
```

## Complete Pipeline Example

```python
from pathlib import Path
from pipeline import (
    ScriptGenerator,
    AudioGenerator,
    TimestampExtractor,
    ActionSyncEngine
)

def run_full_pipeline(prompt: str, output_dir: Path):
    """Run the complete VSpeak pipeline."""
    
    # Stage 1: Script Generation
    print("Stage 1: Generating script...")
    script_gen = ScriptGenerator()
    script, script_latency = script_gen.generate(prompt)
    print(f"  ✓ {script.total_chunks} chunks, {script.total_actions} actions")
    
    # Stage 2: Audio Generation
    print("Stage 2: Generating audio...")
    audio_gen = AudioGenerator(output_dir)
    audio_path, chunk_latencies, stitch_time = audio_gen.generate_audio(script)
    print(f"  ✓ Audio: {audio_path}")
    
    # Stage 3: Timestamp Extraction
    print("Stage 3: Extracting timestamps...")
    ts_extractor = TimestampExtractor()
    timestamps, stt_latency = ts_extractor.extract_timestamps(audio_path)
    print(f"  ✓ {len(timestamps)} word timestamps")
    
    # Stage 4: Action Synchronization
    print("Stage 4: Synchronizing actions...")
    sync_engine = ActionSyncEngine(output_dir)
    synced_actions, sync_latency = sync_engine.sync_actions(script, timestamps)
    actions_path = sync_engine.save_actions_json(synced_actions)
    print(f"  ✓ {len(synced_actions)} actions synced")
    
    return {
        "audio_path": audio_path,
        "actions_path": actions_path,
        "synced_actions": synced_actions
    }

# Run the pipeline
result = run_full_pipeline(
    "Create a Python function that calculates factorial",
    Path("./output")
)

print(f"\nPipeline complete!")
print(f"Audio: {result['audio_path']}")
print(f"Actions: {result['actions_path']}")
```

## Handling Unmatched Triggers

When trigger words are not found in the transcription:

```python
synced_actions, sync_latency = sync_engine.sync_actions(script, timestamps)

# Check for warnings in console output:
# ⚠ Could not find trigger word 'refactor' - using estimate

# Actions are still included but may have estimated timestamps
# Verify critical actions manually:
for action in synced_actions:
    if action.trigger_word == "important_trigger":
        print(f"Verify timestamp: {action.timestamp_ms}ms")
```

## Performance Metrics

Typical synchronization performance:

- **Latency:** 5-50ms for 20-30 actions
- **Match rate:** 90-95% exact trigger word matches
- **Precision:** ±50ms timestamp accuracy

## Debugging Unmatched Actions

```python
import json

# Save timestamps for inspection
with open("timestamps.json", "w") as f:
    json.dump(
        [{"word": ts.word, "start_ms": ts.start_ms} for ts in timestamps],
        f,
        indent=2
    )

# Save actions for comparison
with open("expected_actions.json", "w") as f:
    json.dump(
        [{"trigger": a.trigger_word, "type": a.type} for chunk in script.chunks for a in chunk.actions],
        f,
        indent=2
    )

# Compare to find mismatches
print("Expected triggers:")
for chunk in script.chunks:
    for action in chunk.actions:
        matches = [ts for ts in timestamps if action.trigger_word.lower() in ts.word.lower()]
        print(f"  {action.trigger_word}: {len(matches)} potential matches")
```
